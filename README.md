# Steps 
1. npm install -g ionic
2. ionic -v   5.2.4
3. ionic start storageCrud blank --type=angular
4. cd storageCrud 
5. ionic serve 

# Now for project 
1. ionic g service services/storage
2. npm i @ionic/storage
3. ionic cordova plugin add cordova-sqlite-storage
4. write code 
5. Run with ionic serve
